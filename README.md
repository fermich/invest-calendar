# invest-data-loader

A Clojure library to load and integrate investment data.

## Usage

  - lein run fx all
  - lein run fx diff
###
  - lein run events all
  - lein run events diff
###
  - lein run options all
  - lein run options diff
###
  - lein run stock load-from-dir [dir]
  - lein run stock current-month-diff
